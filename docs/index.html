How to use?

Installation

npm install -s ng-crud-helper

Configuration

You need to add this line:
import{ JSONService } from 'ng-crud-helper/json.service'
and then add to providers and add import HttpModule like this:
import:[HttpModule], providers:[JSONService]

Make requests

Make simple request:
  • doGet(url) -- use this to make get request
  • doGetWithParam(url) -- use this to make get request with one param
  • doPost(url,body) -- use this to make post request
  • doPut(url,param,body) -- use this to make put request
  • doDelete(url,param) -- use this to make delete request
Make requests with ApiKey(When you apikey is sent like url/apikey, if not you need to add manually to url):
  • doGetWithApiKey(url) -- use this to make get request
  • doPostWithApiKey(url,body) -- use this to make post request
  • doPutWithApiKey(url,param,body) -- use this to make put request
  • doDeleteWithApiKey(url,param) -- use this to make delete request
